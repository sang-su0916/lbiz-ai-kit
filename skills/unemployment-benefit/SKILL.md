# 실업급여(구직급여) 계산기 (unemployment-benefit)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **고용보험법 §40** — 구직급여 수급 자격 요건
- **고용보험법 §45** — 구직급여 일액 (이직 전 평균임금의 60%)
- **고용보험법 §50** — 소정급여일수 (가입기간·연령별 매트릭스)
- 상한액 **66,000원/일** (2025년 기준), 하한액 **64,192원/일** (2025년 최저임금 80%, 1일 8시간 기준)

## 자격 요건 (4가지 모두 충족)

1. 이직일 이전 **18개월간 피보험단위기간 180일 이상**
2. **비자발적 이직** — 단, 정당한 사유의 자발적 퇴사도 포함
   - 정당 사유: 단체협약·취업규칙 위반, 임금체불, 근로조건 저하, 통근곤란(왕복 3시간 이상), 결혼·임신·출산·육아, 사업장 이전 등
3. **근로의사·능력 있고** 적극적 구직활동 중
4. 재취업 노력에도 취업하지 못함

## 핵심 공식

```
구직급여 일액 = min(상한액 66,000, max(하한액 64,192, 이직 전 평균임금 일액 × 0.6))
총 지급 추정액 = 구직급여 일액 × 소정급여일수
```

**소정급여일수 매트릭스** (고용보험법 §50 별표):

| 가입기간 \ 연령 | 50세 미만 | 50세 이상 또는 장애인 |
| --------------- | --------- | --------------------- |
| 1년 미만        | 120일     | 120일                 |
| 1년 ~ 3년 미만  | 150일     | 180일                 |
| 3년 ~ 5년 미만  | 180일     | 210일                 |
| 5년 ~ 10년 미만 | 210일     | 240일                 |
| 10년 이상       | 240일     | 270일                 |

## 사용 방법

### `calculate` — 자격 판정 + 일액 + 일수 + 총액

```bash
python3 skills/unemployment-benefit/references/calculator.py calculate \
  --avg-daily-wage 100000 \
  --insured-days 540 \
  --insured-years 5 \
  --age 45 \
  --voluntary no \
  --has-disability no
```

**인자 설명:**

| 인자               | 설명                                 | 필수 |
| ------------------ | ------------------------------------ | ---- |
| `--avg-daily-wage` | 이직 전 평균임금 일액 (원)           | 필수 |
| `--insured-days`   | 이직 전 18개월간 피보험단위기간 (일) | 필수 |
| `--insured-years`  | 고용보험 가입 총 연수                | 필수 |
| `--age`            | 만 나이                              | 필수 |
| `--voluntary`      | 자발적 퇴사 여부 `yes`/`no`          | 필수 |
| `--has-disability` | 장애인 여부 `yes`/`no`               | 필수 |

## 입력 파싱 가이드

사용자가 정보를 일부만 줘도 계산 시도. 정확한 계산 필요 시 추가 정보 요청.

| 사용자 입력                     | 필요 추가 정보                 |
| ------------------------------- | ------------------------------ |
| "5년 다녔는데 실업급여 얼마?"   | 평균임금 일액, 나이, 퇴사 사유 |
| "자발적 퇴사인데 받을 수 있어?" | 정당 사유 해당 여부 추가 확인  |
| "평균임금 월 300만, 3년 근무"   | 일액 환산(÷30) 안내 후 계산    |

## 되묻기 규칙 (정보 부족 시)

1. **임금 정보 부족**: "이직 전 평균임금 일액(또는 월급 ÷ 30)을 알려주세요."
2. **가입기간 부족**: "고용보험 총 가입 연수와 이직 전 18개월간 근무일수를 알려주세요."
3. **퇴사 사유 불명확**: "자발적 퇴사이신가요? 자발적이라면 사유를 알려주세요 (임금체불·통근곤란·결혼·임신 등은 정당 사유로 인정될 수 있습니다)."
4. **나이 미제공**: "만 나이와 장애인 해당 여부를 알려주세요 (수급일수 산정에 영향)."

## 응답 포맷

계산 결과 + 다음 항목 안내:

- 자격 여부 (미달 시 사유 명시)
- 구직급여 일액 (상한·하한 적용 여부 표시)
- 소정급여일수
- 총 지급 추정액
- 산식 (검증용)
- 면책: "총액은 구직활동 인정, 조기재취업수당, 취업촉진수당 등 실제 지급 조건에 따라 달라질 수 있습니다. 정확한 수급 여부는 고용센터에서 확인하세요."

## 알려진 한계

1. **정당 사유 자발적 퇴사 판단 미지원** — `--voluntary yes` 입력 시 자격 없음으로 처리. 정당 사유 해당 여부는 고용센터 심사 사항이므로 별도 안내.
2. **단시간 근로자 비례 미반영** — 1일 8시간 미만 근로자의 하한액 비례 계산은 근로시간 입력이 필요하며, 본 계산기는 8시간 기준 적용.
3. **취업촉진수당 미산정** — 조기재취업수당(잔여일수 50% 이상 남기고 재취업 시 절반 지급), 직업능력개발수당 등 별도 산정.
4. **2026년 하한액** — 최저임금 인상 시 하한액 갱신 필요. 현재 2025년 기준(64,192원) 적용.
5. **피보험단위기간 계산** — 실제 피보험단위기간은 유급처리일 기준이며, 본 계산기는 입력값을 그대로 사용.
