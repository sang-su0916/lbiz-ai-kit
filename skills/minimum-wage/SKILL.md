# 최저임금 위반 체크기 (minimum-wage)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **최저임금법 §5** — 최저임금 고시 (고용노동부장관, 매년 8월 5일 이전 결정·고시)
- **최저임금법 §6** — 최저임금의 효력 (사용자는 최저임금액 이상의 임금 지급 의무)
- **최저임금법 §6③** — 최저임금 산입범위 (2024년 이후 완전 시행)
- 공식 고시: https://www.minimumwage.go.kr

## 2026년 최저임금 (2026.1.1 ~ 2026.12.31 시행)

| 구분                | 금액            |
| ------------------- | --------------- |
| 시급                | **10,320원**    |
| 일급 (8시간)        | **82,560원**    |
| 월 환산액 (209시간) | **2,156,880원** |

- 전년 대비 인상률: **+2.9%** (2025년 10,030원 → 2026년 10,320원)
- 월 209시간 산출: 주 40시간 × 4.345주 + 주휴 8시간 × 4.345주

## 최저임금 산입범위 (2024년 이후 완전 시행)

### 포함 (산입 O)

| 항목                               | 조건                      |
| ---------------------------------- | ------------------------- |
| 기본급                             | 전액                      |
| 정기상여금                         | **매월** 지급하는 것 전액 |
| 복리후생비 (식대·숙박비·교통비 등) | **매월** 지급하는 것 전액 |

### 제외 (산입 X)

| 항목                       | 사유               |
| -------------------------- | ------------------ |
| 분기·반기·연간 지급 상여금 | 매월 지급 아님     |
| 연장근로수당               | 소정근로 외 초과분 |
| 야간근로수당               | 소정근로 외 가산분 |
| 휴일근로수당               | 소정근로 외 가산분 |
| 현물급여                   | 금전 외 지급분     |

## 핵심 공식

```
시간당 실임금 = (기본급 + 월정기상여금 + 월복리후생비) ÷ 소정근로시간
소정근로시간 (주40H) = 40 × 4.345 + 주휴 8 × 4.345 ≈ 209시간/월

위반 조건: 시간당 실임금 < 10,320원
월 부족액 = (10,320 - 시간당 실임금) × 소정근로시간
```

단시간근로자(주 40시간 미만):

```
소정근로시간 = 주간 소정근로시간 × 4.345 + 주휴시간 × 4.345
주휴시간 = 주간 소정근로시간 / 40 × 8  (주 15시간 이상인 경우)
```

## 두 가지 모드

### 1. 위반 체크 (`check`)

월급·주간근로시간·정기상여·복리후생비 입력 → 위반 여부 판정.

```bash
python3 skills/minimum-wage/references/calculator.py check \
  --monthly-wage 2000000 \
  --weekly-hours 40 \
  --regular-bonus 0 \
  --welfare-pay 0
```

### 2. 월 환산액 산출 (`monthly-equivalent`)

임의 시급으로 월 환산액 산출.

```bash
python3 skills/minimum-wage/references/calculator.py monthly-equivalent \
  --hourly-wage 12000 \
  --weekly-hours 40
```

## 입력 파싱 가이드

| 사용자 입력               | 모드               | 누락 시 기본값     |
| ------------------------- | ------------------ | ------------------ |
| "월급 200만, 주40시간"    | check              | bonus=0, welfare=0 |
| "시급 11,000원이면 월급?" | monthly-equivalent | weekly-hours=40    |
| "최저임금 위반인지 봐줘"  | 되묻기 1번         | 월급 또는 시급     |

## 되묻기 규칙 (정보 부족 시)

1. **임금 정보 부족**: "월급(또는 시급)을 알려주세요."
2. **근로시간 불명확**: "주간 소정근로시간은 몇 시간인가요? (기본 40시간)"
3. **정기상여 여부**: "매월 지급되는 정기상여금이 있나요? (분기·연간은 산입 제외)"
4. **복리후생비 여부**: "매월 지급되는 식대·교통비·숙박비 등 복리후생비가 있나요?"

## 응답 포맷

계산 결과 + 다음 항목 안내:

- 시간당 실임금
- 2026년 최저시급 (10,320원)
- 위반 여부 (true / false)
- 시간당 부족액 (위반 시)
- 월 부족액 (위반 시)
- 산식 (검증용)
- 면책: "수습기간·단순노무직 예외 등 특수 케이스는 별도 확인이 필요합니다."

## 알려진 한계

1. **수습기간 90% 감액** — 1년 이상 근로계약 체결 후 최초 3개월은 최저임금의 10% 감액 가능 (단순노무직 제외). 본 계산기는 미반영 — 별도 확인 필요.
2. **단순노무직 예외** — 통계청 한국표준직업분류 대분류 9 (단순노무 종사자)는 수습 감액 적용 불가.
3. **감시·단속 근로자** — 고용부 인가 시 최저임금 적용 특례 가능. 본 계산기 미반영.
4. **동거 친족 근로자** — 최저임금법 적용 제외 대상.
5. **소정근로시간 산정 오차** — 주휴수당 포함 209시간은 주 40시간 기준. 실제 약정 소정근로시간이 다른 경우 직접 입력 권장.

## 관련 스킬

- `wage-base` — 통상임금·평균임금 산정 (최저임금 위반 판정 후 차액 보전 계산)
- `four-insurances` — 최저임금 기준 4대보험 보험료 하한 확인
- `severance-pay` — 최저임금 위반 기간의 퇴직금 소급 계산
