# 부가가치세 계산기 (value-added-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 분기별·반기별 부가세 신고 전 납부세액 점검 (1기 7/25, 2기 익년 1/25 등)
- 간이과세 전환 검토 시 일반 vs 간이 납부세액 비교
- 간이과세 적격 판정 (직전연도 공급대가 1억 400만원 미만 여부)
- 과세기간 공급대가 4,800만원 미만 간이과세자 납부의무 면제 확인
- 업종별 부가가치율 5단계 적용 간이과세 납부세액 산출

## 업종별 부가가치율 (부가가치세법 §63②, 2021.7.1 이후)

| 업종 코드       | 업종                                              | 부가가치율 |
| --------------- | ------------------------------------------------- | ---------- |
| `retail`        | 소매업·재생용 재료수집 및 판매업·음식점업         | 15%        |
| `manufacturing` | 제조업·농업·임업·어업·소화물 전문 운송업          | 20%        |
| `lodging`       | 숙박업                                            | 25%        |
| `construction`  | 건설업·운수 및 창고업·정보통신업                  | 30%        |
| `service`       | 금융·보험·부동산·전문과학기술·사업지원·기타서비스 | 40%        |

## 핵심 공식

```
─ 일반과세 (§30) ──────────────────────────────────────────────
매출세액 = 매출 공급가액 × 10%
매입세액 = 매입 공급가액 × 10%
납부세액 = 매출세액 - 매입세액   (환급: 음수 가능)

─ 간이과세 (§63②) ─────────────────────────────────────────────
납부세액 = 공급대가 × 업종별 부가가치율 × 10%
         - 매입세액공제 (매입공급가액 × 0.5%)

─ 간이과세 적격 (§61) ─────────────────────────────────────────
  · 개인사업자 (법인 불가)
  · 직전연도 공급대가 < 1억 400만원
  · 간이과세 배제업종 해당 없음

─ 납부의무 면제 (§69) ─────────────────────────────────────────
해당 과세기간 공급대가 < 4,800만원 → payable_vat = 0
```

## CLI 사용법

### `general` — 일반과세 납부세액

```bash
python3 skills/value-added-tax/references/calculator.py general \
  --sales-supply 100000000 \
  --purchase-supply 60000000
```

기타 플래그: `--zero-rate-sales` (영세율 매출), `--exempt-sales` (면세 매출).

### `simplified` — 간이과세 납부세액

```bash
python3 skills/value-added-tax/references/calculator.py simplified \
  --supply-price 80000000 \
  --industry retail
```

기타 플래그: `--purchase-supply` (매입 공급가액 — 0.5% 공제), `--prior-year-supply-price` (직전연도 공급대가).

### `eligibility` — 간이과세 적격 판정

```bash
python3 skills/value-added-tax/references/calculator.py eligibility \
  --prior-year-supply-price 90000000
```

플래그: `--is-corporate` (법인 flag), `--is-restricted-industry` (배제업종 flag).

### `compare` — 일반 vs 간이 비교

```bash
python3 skills/value-added-tax/references/calculator.py compare \
  --sales-supply 100000000 --purchase-supply 60000000 \
  --supply-price 110000000 --industry retail
```

출력: 양쪽 납부세액 + `difference` + `recommended` + `rationale`.

## 되묻기 규칙 (정보 부족 시)

1. **공급가액 vs 공급대가**: "말씀하신 금액이 공급가액(VAT 제외)인가요, 공급대가(VAT 포함)인가요? 일반과세는 공급가액, 간이과세는 공급대가 기준입니다."
2. **업종 분류**: "구체적인 업종을 알려주세요. 소매/음식점(15%), 제조/농림어업(20%), 숙박(25%), 건설/운수/정보통신(30%), 서비스/금융(40%) 중 어디에 해당하나요?"
3. **법인 여부**: "개인사업자인가요 법인인가요? 법인은 매출 규모와 무관하게 간이과세 불가입니다."
4. **매입세액 공제 가능 여부**: "매입 시 세금계산서를 수령했나요? 간이과세자로부터 매입·영수증 매입은 매입세액공제 대상이 아닐 수 있습니다."
5. **배제업종 확인**: "간이과세 배제업종(광업·제조업 일부·도매업·부동산매매업 등) 해당 여부 확인 필요합니다."

## 도메인 특수 규칙

- **영세율 vs 면세**: 영세율(수출 등)은 매출세액 0원이지만 매입세액 환급 가능. 면세(식료품·교육·의료 등)는 애초에 부가세 대상 아님 — 매입세액 환급도 불가.
- **간이과세자 매입세액공제**: 일반과세자와 달리 매입공급가액 × 0.5% (업종무관). 세금계산서·신용카드매출전표 등 적격증빙 수령분만 공제.
- **납부의무 면제 vs 신고의무**: §69 면제는 납부세액만 0원. 신고의무는 존재 (무신고 가산세 방지).
- **일반과세 환급**: 납부세액이 음수면 환급. 간이과세자는 환급 불가.
- **지방소비세 별도**: 본 계산기는 부가세(국세)만 산출. 지방소비세는 부가세의 25.3% (2026 기준, 세입예산 기준) 별도.
- **corporate-tax-interim-payment·preliminary-closing 시너지**: 분기 가결산 시 매출·매입 산정 → 본 스킬로 예정 부가세 확인 → 법인세 중간예납 현금흐름 계획.

## 알려진 한계

1. **의제매입세액공제 미반영** — 음식점업·제조업 등 면세농산물 매입 시 의제매입세액공제 별도 규정.
2. **공통매입세액 안분 미지원** — 과세·면세 겸영 사업자의 공통매입세액 안분계산 (공급가액 비율 등) 미반영.
3. **가산세 미계산** — 무신고·과소신고·납부지연 가산세 미포함.
4. **재고매입세액·대손세액공제 미반영** — 간이→일반 전환 시 재고매입세액공제, 대손 발생 시 대손세액공제 별도.
5. **간이과세 배제업종 자동 판정 불가** — `--is-restricted-industry` flag 수동 입력.
6. **지방소비세 미포함** — 실제 납부 총액은 부가세 + 지방소비세.

## 관련 스킬

- `income-tax` — 개인 종합소득세 (부가세는 간접세, 종소세는 직접세)
- `corporate-tax-interim-payment` — 법인세 중간예납 (부가세 예정신고와 동일 시기대 현금부담 점검)
- `preliminary-closing` — 분기·반기 가결산으로 매출·매입 산정 → 본 스킬로 예정 부가세 추정
- `omsc` — 본 스킬은 OMSC scaffold 기반 세무 도메인 스킬
