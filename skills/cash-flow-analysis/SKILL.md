# 현금흐름표 분석 스킬 (cash-flow-analysis)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 언제 사용

- 현금흐름표의 영업(OCF)·투자(ICF)·재무(FCF_fin) 세 활동 부호 조합으로 기업 생애주기 판별
- 장부이익(순이익) 대비 실제 현금 창출력(영업CF) 품질 점검
- 잉여현금흐름(FCF = OCF - CapEx)으로 자체 투자여력 확인
- 재무비율 진단만으로 놓치기 쉬운 한계기업 조기 경보 (OCF 음수·재무CF 지속 +)
- 대출·M&A 실사 전 현금 건전성 1차 스크리닝

## 법령·근거

- **일반기업회계기준 제5장** — 현금흐름표 (영업·투자·재무 활동 분류)
- **K-IFRS 제1007호** — 현금흐름표 (Statement of Cash Flows)
- 영업활동: 매출·매입·인건비·법인세 등 주된 수익창출 활동에서 발생하는 현금흐름
- 투자활동: 유형·무형자산 취득·처분, 금융상품 투자 등
- 재무활동: 차입·상환, 유상증자, 배당, 자기주식 거래 등

## 8가지 CF 패턴 분류

영업(O)·투자(I)·재무(F) 세 활동 부호 조합 (2³ = 8가지):

| 패턴         | O   | I   | F   | 유형            | 해석                                                      |
| ------------ | --- | --- | --- | --------------- | --------------------------------------------------------- |
| 안정형       | +   | -   | -   | 성숙기 우량기업 | 영업이익으로 투자·배당/차입금 상환 — 전형적 우량기업      |
| 성장형       | +   | -   | +   | 성장기 기업     | 영업흑자 + 추가 자본조달로 투자 확대                      |
| 구조조정형   | +   | +   | -   | 구조조정기      | 영업흑자 + 자산매각으로 부채 상환                         |
| 유동성비축형 | +   | +   | +   | 드문 케이스     | 자산매각·조달까지 동시 — 대규모 M&A 준비 또는 이례적 상황 |
| 확장형       | -   | -   | +   | 초기창업·확장기 | 영업적자를 외부자금으로 보전하며 투자 확대                |
| 쇠퇴형       | -   | +   | -   | 쇠퇴기          | 영업적자를 자산매각으로 충당하며 부채 상환 — 위험         |
| 재무위기형   | -   | -   | -   | 재무위기        | 모든 활동에서 현금유출만 — 즉시 위기 대응 필요            |
| 심각형       | -   | +   | +   | 한계기업        | 영업적자 + 자산매각 + 외부조달 동시 — 생존 한계           |

## 영업CF 품질 지표

| 지표              | 공식             | 해석                                                |
| ----------------- | ---------------- | --------------------------------------------------- |
| OCF / 순이익 비율 | OCF ÷ 당기순이익 | 1.0 이상 양호, 0.5 미만 분식 징후 (장부이익 > 현금) |
| OCF / 매출 비율   | OCF ÷ 매출       | 업종 평균 대비 비교 — 10% 이상 양호                 |
| 잉여현금흐름(FCF) | OCF - CapEx      | + 이면 자체 투자여력, - 지속되면 외부 의존          |

## 위험 플래그 규칙

- **OCF < 0**: 영업활동 자체가 현금유출 — 수익구조 재점검 시급
- **OCF/순이익 < 0.5**: 장부이익 대비 실제 현금 창출 부족 — 매출채권·재고 급증 확인
- **FCF < 0 지속**: 투자 대비 자체 현금 부족 — 외부 의존도 확인
- **재무CF 지속 (+)**: 외부 자본 의존 — 구조 개선 필요
- **패턴 (-,-,-)**: 긴급 유동성 대책 필요 — 자산매각·차입·자본조달 검토
- **패턴 (-,+,+)**: 한계기업 징후 — 영업·재무 동시 악화

## CLI 사용법

### `analyze` — 현금흐름 종합 분석

```bash
python3 skills/cash-flow-analysis/references/calculator.py analyze \
  --operating-cf 80000000 \
  --investing-cf -50000000 \
  --financing-cf -20000000 \
  --net-income 70000000 \
  --capex 50000000 \
  --revenue 500000000
```

출력 JSON 주요 필드:

- `summary.operating_cf` / `investing_cf` / `financing_cf` / `net_cash_change`
- `free_cash_flow`: OCF - CapEx (CapEx 입력 시)
- `cf_pattern`: 8가지 분류 키워드 (안정형·성장형·쇠퇴형 등)
- `pattern_description`: 패턴 해석 문장
- `quality_indicators.ocf_to_net_income_ratio` / `ocf_to_revenue_ratio` / `quality_interpretation`
- `flags`: 위험 경보 배열
- `recommendations`: 권고사항 배열
- `disclaimer`: 면책 문구

## 되묻기 규칙

1. **부호 확인**: "투자·재무 현금흐름은 유출이면 마이너스(-)로 입력해 주세요."
2. **CapEx 범위**: "자본적지출(CapEx)은 일반적으로 유형자산 취득액을 사용합니다. 투자CF 전체가 아니라 CapEx만 입력해 주세요."
3. **연결 vs. 별도**: "연결 현금흐름표인가요 별도 기준인가요? 자회사 자금흐름 포함 여부가 달라집니다."
4. **회계기준**: "K-IFRS 기준인가요 일반기업회계기준인가요? 이자·배당 분류 범주가 다를 수 있습니다."
5. **기간**: "연간(12개월) 기준인가요, 반기·분기 기준인가요? 비율 해석에 영향이 있습니다."

## 도메인 특수 규칙

- **단년 기준**: 본 스킬은 1개 회계기간 분석 — 3-5년 추세는 별도 계산 필요.
- **이자·배당 분류 주의**: K-IFRS는 영업/재무 선택 가능, 일반기업회계기준은 고정 — 분류 차이로 OCF 수치 달라질 수 있음.
- **CapEx 정의**: 본 스킬은 CapEx를 별도 입력받음. "투자CF = -CapEx"가 아닌 경우(처분·무형자산 포함)가 많으므로 주의.
- **업종 편차**: 금융업·유틸리티·스타트업은 CF 패턴 해석이 일반 제조업과 다름.

## 응답 포맷

계산 결과를 다음 순서로 제시하세요.

1. CF 패턴 + 유형 (굵게) + 한 줄 설명
2. 영업/투자/재무 CF 숫자 요약 표
3. 영업CF 품질 지표 (OCF/순이익, OCF/매출, FCF)
4. 위험 플래그 (있을 때만)
5. 권고사항 (bullet)
6. 면책 문구

**면책**: 본 분석은 현금흐름표 단년 참고치이며, 업종·회계기준(K-IFRS vs. 일반기업회계기준)·이자배당 분류 방침에 따라 해석이 달라집니다. 실제 투자·대출·M&A 판단은 회계사·재무전문가 검토가 필요합니다.

## 알려진 한계

1. **단년 기준**: 3-5년 추세 분석 미지원. 한계기업 정식 정의(연속 이자보상 1배 미만)는 `financial-diagnosis` 참조.
2. **CapEx 단일 입력**: 유형·무형자산 구분, 감액손실 환입 등 세부 항목 미지원.
3. **이자·배당 분류 고정**: K-IFRS 선택권(이자를 재무로 분류 등) 자동 전환 없음.
4. **업종 기준치 단일**: 제조업 기준 — 금융·SaaS·건설업 별도 해석 필요.
5. **간접법·직접법 구분 없음**: 입력값은 최종 OCF·ICF·FCF_fin만 사용 — 조정 세부항목 분석 미지원.

## 관련 스킬

- `financial-diagnosis` — 재무비율 종합진단 (CF와 상호 보완)
- `financial-ratio` — 유동성·안정성·수익성·활동성 비율
- `financial-statement-trend` — 3-5년 추세 분석 (CF 연속 패턴 점검)
- `break-even` — 손익분기점 분석
- `omsc` — 본 스킬은 OMSC scaffold로 생성된 경영 도메인 스킬
