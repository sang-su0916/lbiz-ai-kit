# 근로계약서 검토 (labor-contract-review)

**반드시 계약서 전문을 제공하거나 사진을 업로드하세요.** 텍스트 발췌 또는 불완전한 정보로는 정확한 검토 불가능합니다.

## 법령 근거

- **근로기준법 §17** — 근로계약의 명시 (위반 시 500만원 이하 벌금)
- **근로기준법 §17②** — 서면 교부 의무 (미교부 시 500만원 이하 벌금)
- **근로기준법 §17조의2** — 단시간근로자 추가 명시 (4주간 1주 15시간 미만)
- **근로기준법 §20** — 손해배상액 예정 금지 (무효)
- **근로기준법 §21** — 전차금 상계 금지
- **근로기준법 §22** — 강제저금·저축 금지
- **근로기준법 §66** — 연소자 친권자 동의 + 가족관계증명서 비치
- **근로기준법 §69** — 연소자 근로시간 제한 (1일 7시간, 주 35시간)
- **근로기준법 §96·97** — 단체협약·취업규칙 우위

## 7가지 필수 명시사항 체크리스트

근로계약서에 **반드시** 포함되어야 할 항목:

| #   | 항목                                          | 명시 내용                                                                 | 체크 |
| --- | --------------------------------------------- | ------------------------------------------------------------------------- | ---- |
| 1   | **임금**                                      | 구성항목(기본급·수당), 계산방법, 지급방법(현금/계좌), 지급일              | ☐    |
| 2   | **소정근로시간**                              | 1일 근로시간, 주 근로시간 (예: 1일 8시간, 주 40시간)                      | ☐    |
| 3   | **휴일**                                      | 주휴일(예: 일요일), 공휴일 적용 여부                                      | ☐    |
| 4   | **연차유급휴가**                              | 연 15일(2개월 미만 1개월당 1일), 지급시기(퇴직시 미사용분 수당 지급 여부) | ☐    |
| 5   | **취업장소·종사업무**                         | 근무지(본사/지점/현장), 담당 업무 명확히                                  | ☐    |
| 6   | **근로일·근로시간** (단시간근로자만 §17조의2) | 4주간 근무 일정, 시간 명시 (주 15시간 미만)                               | ☐    |
| 7   | **계약기간** (기간제 근로자만)                | 시작일, 종료일, 갱신 조건(있으면)                                         | ☐    |

**평가 기준**:

- ✅ **충분**: 항목이 명확하고 구체적으로 기재
- ⚠️ **미흡**: 항목이 있으나 모호하거나 부분적 (예: "임금은 회사 규정에 따름")
- ❌ **누락**: 항목이 없음 (위반 ❌)

## 불공정 조항 스캔 (위법 여부)

### 1. 손해배상액 예정 (§20 위반) ❌

**금지 패턴**:

- "1년 이내 퇴사 시 위약금 XXXX원 징수"
- "계약기간 중도 해지 시 교육비/이직금 배상"
- "자격증 미취득 시 교육비 전액 상환"

**평가**:

- ❌ 발견 → "무효 조항. 근로자는 이를 거부할 권리 있음"
- ✅ 미발견 → "양호"

### 2. 강제저금·저축 (§22 위반) ❌

**금지 패턴**:

- "월급 중 5% 강제 저금"
- "주택자금·교육자금 선납금 징수 후 월급에서 차감"
- "보증금 명목 선입금"

**평가**:

- ❌ 발견 → "불법. 근로자 자발적 동의도 무효"
- ✅ 미발견 → "양호"

### 3. 전차금 상계 (§21 위반) ❌

**금지 패턴**:

- "기존 회사에서 받은 선수금/대출금을 이번 회사 임금에서 차감"
- "이전 계약 위반금을 당월 임금에서 공제"

**평가**:

- ❌ 발견 → "불법. 임금 구성 항목 외 공제 금지"
- ⚠️ 의심 → "해당 항목의 구체적 내용 확인 필요"
- ✅ 미발견 → "양호"

### 4. 과도한 수습기간 ⚠️

**기준** (1년 이상 계약 기간제 근로자 제외 단순노무 직종):

- 3개월 초과 수습기간 + 최저임금 90% 감액 → 위법 가능성

**체크**:

- 수습기간이 "3개월"라고 명시 → 양호
- 수습기간이 "6개월"이상 + 임금 90% → ⚠️ "미흡. 법정 수습기간 3개월이 상한"
- 수습기간 없음 → ✅ 양호

### 5. 과도한 비용 부담 ⚠️

**의심 패턴**:

- "연간 교육비 부담 시 ○년 이상 재직 의무"
- "자격증 취득비 지원 시 3년 근무 서약"

**평가**:

- ⚠️ "조건부 혜택은 법적으로 문제될 수 있으니 협의 권장"
- 일반적 교육/자격증 지원은 선의로 해석 가능

## 계약 형태별 추가 체크

### 기간제 근로자

- ✅ 계약기간(시작일/종료일) 명시
- ⚠️ "갱신 조건" 모호 → "자동 갱신인지 재계약인지 명확히"
- ⚠️ 부정기간계약(기간 미정) → "1년 이내로 제한되어야 함"

### 단시간근로자 (4주 1주 15시간 미만, §17조의2)

- ✅ "4주간 근무일정 및 시간" 명시
- ❌ 일정이 모호하거나 "월별로 결정" → 명시사항 부재

### 기간제근로자 + 단시간근로자 (예: 행사성 용역 3개월, 주 12시간)

- 두 규정 모두 적용 → 기간 + 일정 모두 명시 필수

### 연소자 (18세 미만)

- ❌ 친권자/후견인 동의서 미첨부 → 무효
- ❌ 가족관계증명서 비치 미확인 → 위반
- ❌ 근로시간 1일 8시간 초과 또는 주 40시간 초과 → 위반 (§69)
  - **정상**: 1일 7시간 이하, 주 35시간 이하
  - **초과**: ⚠️ "근로기준법 위반. 교정 필요"

## 검토 워크플로

### STEP 1: 계약 형태 판정

사용자에게 되묻기 (필요시):

```
1. 계약 형태: 정규직 / 기간제 / 단시간근로자 / 일용직 / 기타?
2. 계약기간: 무기한(정규) / 기간 정함(예: 2026년 1월 1일 ~ 3월 31일)?
3. 근로자 나이: 18세 이상 / 미만?
4. 근무 형태: 주 40시간 이상(정규) / 15시간 미만(단시간)?
```

### STEP 2: 필수 명시사항 7개 체크

제공된 계약서 전문을 순회 → 위 표 7개 항목별로 ✅/⚠️/❌ 매핑

### STEP 3: 불공정 조항 스캔

위 5가지 금지 패턴(손해배상·강제저금·전차금·과도한 수습·비용 부담) 검색

### STEP 4: 계약 형태별 추가 체크

기간제/단시간/연소자 등 해당 조건 검증

### STEP 5: 응답 포맷

다음 구조로 응답:

```markdown
## 검토 결과

### 필수 명시사항 (근로기준법 §17)

| 항목            | 상태 | 내용                                  |
| --------------- | ---- | ------------------------------------- |
| 1. 임금         | ✅   | 기본급 250만원, 매월 25일 계좌이체    |
| 2. 소정근로시간 | ⚠️   | 1일 8시간 명시, 단 주 근로시간 미표기 |
| ...             | ...  | ...                                   |

**누락 항목**: [항목 번호 나열], 보완 필요

### 불공정 조항 검토

| 조항            | 판정    | 비고                                          |
| --------------- | ------- | --------------------------------------------- |
| 손해배상액 예정 | ✅ 없음 | -                                             |
| 강제저금/저축   | ❌ 발견 | "월급 5% 강제저금" → **무효. 즉시 삭제 요청** |
| ...             | ...     | ...                                           |

**위반 항목**: [조항] → 법정 대응 방법 안내

### 계약 형태별 검토

(해당사항이 있으면 기간제/단시간/연소자별 추가 체크)

### 최종 평가

- **완전**: 필수 명시사항 7개 모두 충분 + 불공정 조항 없음 → "즉시 서명 가능"
- **보완**: 미흡/누락 항목 있음 → "사업주 또는 인사팀에 보완 요청"
- **위반**: 불공정 조항 발견 → "해당 조항 삭제 또는 서명 거부 권장"

### 다음 단계 권고

1. [구체적 보완 사항]
2. [분쟁 발생 시 대응 방법]
3. [추가 확인 필요 항목 (필요시)]

### 면책

이 검토는 **근로기준법 기준 1차 자동 점검**입니다. 실제 분쟁·소송 대응은 **공인노무사(중앙회: nllb.or.kr) 또는 근로감시관(고용노동부)** 상담을 권장합니다.

- 단체협약·취업규칙이 있으면 그 조건이 근로기준법보다 유리하면 우선 적용됨 (§96·97)
- 재택근무·선택근로제 등 특수 고용형태는 별도 검토 필요
```

## 되묻기 규칙 (정보 부족 시)

사용자가 불완전한 정보를 제공한 경우:

| 부족 정보                 | 재질문                                                          |
| ------------------------- | --------------------------------------------------------------- |
| 계약서 텍스트만 부분 제공 | "전체 계약서를 업로드하거나 [누락 섹션] 내용을 알려주세요."     |
| 계약 형태 불명            | "정규직 / 기간제 / 단시간근로자 / 일용직 중 어디에 해당하나요?" |
| 계약기간 불명             | "시작일 / 종료일 또는 '무기한'인지 알려주세요."                 |
| 근로자 나이 불명          | "근로자가 18세 미만인가요?" (연소자 조항 적용 판정용)           |
| 임금 정보 불완전          | "임금 구성(기본급, 수당)과 지급일을 구체적으로 알려주세요."     |
| 소정근로시간 불명         | "1일 근로시간과 주 근로시간을 알려주세요."                      |

**원칙**: 정보 부족으로 검토가 불가능하면 "보충 정보 요청" → 완전한 정보 수집 후 재검토.

## 입력 파싱 가이드

사용자 제공 방식별 처리:

| 입력 방식                                         | 처리 방법                              |
| ------------------------------------------------- | -------------------------------------- |
| 전체 계약서 텍스트 복사                           | 전체 스캔 (STEP 2~5 수행)              |
| 계약서 사진/PDF                                   | OCR/이미지 해석 후 텍스트 추출 → 스캔  |
| "조항별 요약" (예: "임금 250만, 주 40시간")       | 정보 부족 → 재질문 또는 전체 문서 요청 |
| "조항 중 일부만" (예: "강제저금 있는지 확인해줘") | 특정 조항만 검토 불가 → 전체 문서 필요 |

## 알려진 한계

1. **특수 고용형태 미지원**: 프리랜서·특수형태근로종사자(배달·택시·외판원 등)는 고용관계 자체가 다름. 별도 법령 적용.

2. **취업규칙·단체협약 충돌 미분석**: 계약서만으로는 상위 규범(§96·97)과의 우선순위 판정 불가. 취업규칙 함께 제공 필요.

3. **산업·직종별 특수 규정 미포함**:
   - 감시·단속적 근로자 (§63)
   - 관리·감독 근로자 (§63)
   - 외항해선원 등 특수 직종
     → 별도 검토 필요

4. **해석 쟁점 미반영**: 판례·행정 해석이 분분한 경우 (예: "통상임금 범위") 일반론만 제시. 최신 판결/행정 지침 확인 권장.

5. **국제 원정 근로 미지원**: 파견·해외 현지채용 등의 경우 해외법, 파견법, 파견근로자보호법 등 추가 확인 필요.

## 관련 스킬

- `wage-base` — 임금 구성 항목(기본급·통상임금·평균임금) 산정 기준
- `severance-pay` — 퇴직금 산정 (계약서 "연차유급휴가" 항목과 연계)
- `annual-leave` — 연차휴가 부여 방식 (계약서 명시 여부 검증)
- `four-insurances` — 4대보험 가입 의무 (계약서에 명시될 사항)

## 표준근로계약서 양식 참조

실무에서 사용되는 표준 양식:

1. **고용노동부 표준 근로계약서**: https://www.moel.go.kr (공식 템플릿)
2. **한국경영자총협회 양식**: 대기업 표준
3. **중소기업중앙회 양식**: 중소기업용

정기적으로 업데이트되므로 최신 버전 다운로드 후 비교 권장.
