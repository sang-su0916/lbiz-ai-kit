# 종합소득세 계산기 (income-tax)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

| 조문            | 내용                                                         | 출처                                                        |
| --------------- | ------------------------------------------------------------ | ----------------------------------------------------------- |
| 소득세법 §55    | 종합소득세 기본세율 (8단계 누진세율, 2023년 귀속분부터 현행) | [국세청 nts.go.kr](https://www.nts.go.kr) (2026-04-14 조회) |
| 소득세법 §59의2 | 세액공제 (근로·자녀·연금계좌 등) — 본 계산기 **미반영**      | law.go.kr                                                   |
| 지방세법 §92    | 지방소득세 = 종합소득세 산출세액의 10%                       | law.go.kr                                                   |

## 핵심 공식

```
산출세액     = 과세표준 × 세율 - 누진공제
지방소득세   = 산출세액 × 10% (별도, 지방세)
최종부담세액 = 산출세액 + 지방소득세 - 세액공제 - 기납부세액
```

**과세표준 = 종합소득금액 - 종합소득공제** (본 계산기는 과세표준을 직접 입력받음)

## 세율표 (소득세법 §55, 8단계)

| 과세표준 구간                             | 세율 | 누진공제액 |
| ----------------------------------------- | :--: | ---------: |
| 14,000,000원 이하                         |  6%  |          0 |
| 14,000,000원 초과 ~ 50,000,000원 이하     | 15%  |  1,260,000 |
| 50,000,000원 초과 ~ 88,000,000원 이하     | 24%  |  5,760,000 |
| 88,000,000원 초과 ~ 150,000,000원 이하    | 35%  | 15,440,000 |
| 150,000,000원 초과 ~ 300,000,000원 이하   | 38%  | 19,940,000 |
| 300,000,000원 초과 ~ 500,000,000원 이하   | 40%  | 25,940,000 |
| 500,000,000원 초과 ~ 1,000,000,000원 이하 | 42%  | 35,940,000 |
| 1,000,000,000원 초과                      | 45%  | 65,940,000 |

## CLI 사용법

### `calculate` — 산출세액 + 지방소득세

```bash
python3 skills/income-tax/references/calculator.py calculate \
  --taxable-income 50000000
```

출력: 산출세액, 지방소득세, 한계세율, 누진공제, 실효세율, 산식.

### `effective-rate` — 한계세율 vs 실효세율 빠른 조회

```bash
python3 skills/income-tax/references/calculator.py effective-rate \
  --taxable-income 100000000
```

출력: 한계세율(구간세율) + 국세 실효세율 + 지방세 포함 총실효세율.

## 되묻기 규칙 (정보 부족 시)

1. **과세표준/소득 혼동**: "종합소득금액(총소득)인가요, 아니면 공제를 뺀 과세표준인가요? (과세표준 = 종합소득금액 − 종합소득공제)"
2. **공제 정보 부족**: "인적공제(부양가족 수), 국민연금 납입액, 건강보험료 등 소득공제 금액을 알려주시면 과세표준을 계산해드립니다. 모르시면 '과세표준만 입력'으로 진행할 수 있습니다."
3. **소득 종류 불명**: "근로소득인가요, 사업소득인가요, 기타소득인가요? 분리과세 대상(이자·배당 2천만 이하 등)이라면 종합소득세 대상이 아닙니다."
4. **기준년도**: "2023년 귀속분부터 현행 세율표가 적용됩니다. 2022년 이전은 다른 세율이므로 확인이 필요합니다."

## 도메인 특수 규칙

- **지방소득세 별도 부과**: 산출세액 × 10%, 지방자치단체에 납부 (지방세)
- **분리과세 소득 제외**: 이자·배당 2천만원 이하, 복권 당첨금, 퇴직소득, 양도소득 등은 종합과세 대상 아님
- **연말정산 vs 종합소득세 신고**: 근로소득만 있는 경우 회사 연말정산으로 종결, 사업·기타소득 병행 시 5월 종합소득세 신고 필요
- **간이세액표**: 월 급여 원천징수는 간이세액표(별표2) 기준 — 본 계산기와 별개

## 응답 포맷

계산 결과에 다음 항목을 포함하세요.

- 과세표준 (입력값 재확인)
- 한계세율 + 누진공제액 (구간 설명)
- 산출세액 (국세)
- 지방소득세 (10%)
- 총 세부담액
- 실효세율 (국세 기준 / 총세부담 기준)
- 산식 (검증용)
- 면책 문구

**면책**: 본 계산은 소득세법 §55 기본세율표 기준 산출세액입니다. 세액공제(근로·자녀·연금계좌), 기납부세액, 감면, 분리과세 특례는 미반영입니다. 최종 세액은 홈택스(hometax.go.kr) 또는 세무사 확인이 필요합니다.

## 알려진 한계

1. **종합소득공제·세액공제 미산정** — 인적공제·연금보험료공제·특별소득공제 등은 사전에 별도 계산해 과세표준을 만든 뒤 입력해야 합니다. 근로소득세액공제·자녀세액공제·연금계좌세액공제도 미반영.
2. **연말정산 특례 미지원** — 근로소득자 간이세액표 원천징수, 연말정산 환급·추징, 중도퇴사 정산 로직 없음.
3. **분리과세·감면 제외** — 이자·배당 2천만 이하 분리과세, 주택임대 분리과세(2천만 이하), 농어촌소득세 감면 등 특례 미반영.
4. **종합소득금액 산정 미포함** — 사업소득 필요경비, 근로소득공제(근로소득금액 산정), 기타소득 필요경비 등은 본 계산기 범위 밖. 과세표준이 확정된 상태에서만 사용.
5. **퇴직소득세·양도소득세 별도** — 퇴직·양도소득은 분류과세로 별도 세율표 적용 (퇴직금은 `severance-pay` 참고, 향후 `capital-gains-tax` 예정).

## 관련 스킬

- `severance-pay` — 퇴직금 자체는 분류과세(퇴직소득세 별도), 본 계산과 합산하지 않음
- `four-insurances` — 국민연금·건강보험·고용보험료는 종합소득공제 항목 (과세표준 산정 베이스)
- `wage-base` — 근로소득 과세표준 산정 시 비과세 식별용 베이스
- `capital-gains-tax` _(예정)_ — 양도소득세 별도 세율표
- `omsc` — 본 스킬은 OMSC scaffold로 생성된 첫 세무 도메인 스킬
