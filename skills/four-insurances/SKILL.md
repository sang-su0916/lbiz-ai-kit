# 4대보험료 계산기 (four-insurances)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **국민연금법 §88** — 보험료율 9% (사업주 4.5% + 근로자 4.5%)
- **국민건강보험법 §69, §73** — 건강보험료율 7.09% (사업주 3.545% + 근로자 3.545%)
- **노인장기요양보험법 §9** — 장기요양보험료 = 건강보험료 × 12.95%
- **고용보험법 §49, §70** — 실업급여 1.8% / 고용안정·직업능력개발 사업주 추가부담
- **산업재해보상보험법 §13** — 산재보험 사업주 100% 부담, 업종별 요율 고시

> 요율 변경 시 `references/calculator.py` 상단 RATES dict 수정.

## 두 가지 모드

### 1. 4대보험료 계산 (`calculate`)

월 보수 기준 항목별 근로자·사업주 부담금 산정.

```bash
python3 skills/four-insurances/references/calculator.py calculate \
  --monthly-wage 3000000 \
  --company-size under_150 \
  --industry-rate 0.0143
```

`--company-size` 선택지:

| 값                  | 설명                          | 고용안정·직능 요율 |
| ------------------- | ----------------------------- | ------------------ |
| `under_150`         | 150인 미만                    | 0.25%              |
| `over_150_priority` | 150인 이상 + 우선지원대상기업 | 0.45%              |
| `over_150`          | 150인 이상~1,000인 미만       | 0.65%              |
| `over_1000`         | 1,000인 이상 또는 국가·지자체 | 0.85%              |

### 2. 세전→실수령 추정 (`compare-with-net`)

세전 월급에서 4대보험 근로자 부담분만 차감한 세후 추정액 (소득세·지방소득세 미적용, 단순 참고용).

```bash
python3 skills/four-insurances/references/calculator.py compare-with-net \
  --gross-monthly-wage 3000000
```

## 입력 파싱 가이드

| 사용자 입력               | 모드             | 기본값 처리                    |
| ------------------------- | ---------------- | ------------------------------ |
| "월급 300만, 50인 사업장" | calculate        | industry-rate=0.0143           |

## 되묻기 규칙 (정보 부족 시)

1. **임금 정보 부족**: "월 보수(세전 급여)를 알려주세요."
2. **사업장 규모 불명**: "상시 근로자 수가 150인 미만인가요?"
3. **산재요율 불명**: "업종을 알려주시면 산재요율을 적용합니다. (모르시면 평균 1.43% 사용)"
4. **일용직·단기**: "4대보험 가입 여부 및 월 근로일수를 확인해 주세요."

## 응답 포맷

계산 결과 + 다음 항목 안내:

- 항목별 근로자 부담 / 사업주 부담 / 합계
- 근로자 총 공제액 / 사업주 총 부담액 / 전체 합계
- 산식 (검증용)
- 면책: "고용형태(일용직·외국인·대표이사 등), 보수 외 소득, 업종 변경 등에 따라 실제 금액이 달라질 수 있습니다."

## 알려진 한계

1. **소득세 미산정** — 실수령 추정 시 소득세·지방소득세는 간이세액표 별도 적용 필요.
2. **일용직·프리랜서 제외** — 월 60시간 미만 단시간·일용직은 가입 제외 요건 별도 확인.
3. **대표이사·사용자** — 건강보험은 가입 가능, 고용·산재는 적용 제외 원칙 (예외 있음).
4. **장기요양 정산** — 건강보험료 × 12.95%는 보험료 고지 기준. 정산 시 연말 보수총액 기준 재계산.
5. **2026.7 요율 변경** — 국민연금 기준소득월액 상·하한 매년 7월 갱신. 변경 후 RATES dict 업데이트 필요.

## 관련 스킬

- `unemployment-benefit` — 실업급여 수급 자격 및 금액
