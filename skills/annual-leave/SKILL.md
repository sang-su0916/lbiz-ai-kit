# 연차유급휴가 계산기 (annual-leave)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **근로기준법 §60** — 연차유급휴가 발생 및 가산
- **근로기준법 §61** — 연차유급휴가 사용촉진 (이행 완료 시 사용자 미사용수당 지급의무 면제)
- 4주간 1주 평균 15시간 미만 단시간 근로자 → 적용 제외 (§18③)

## 핵심 공식

```
[1년 미만]
  1개월 개근 시 1일 발생 → 입사 후 n개월 개근 = n일 (최대 11일)

[1년 이상, 출근율 80% 이상]
  기본 15일

[3년 이상 근속 가산]
  가산일수 = floor((근속연수 - 1) / 2)   ← 3년차부터 1일씩 추가
  상한: 25일
  (3년→16일, 5년→17일, 7년→18일 … 21년 이상→25일)

[미사용 연차수당]
  수당 = 통상임금 1일분 × 미사용일수
  단, 연차사용촉진(§61) 완료 시 사용자 지급의무 면제 → 0원

[회계연도 기준 첫해 비례]
  비례발생일수 = round(입사일~회계연도말 일수 / 365 × 15, 2)
```

## 두 가지 모드

### 1. 발생일수 산정 (`entitlement`)

입사일·기준일·기준유형·출근율 입력 → 법정 발생일수 산정.

```bash
python3 skills/annual-leave/references/calculator.py entitlement \
  --hire-date 2020-04-13 \
  --base-date 2026-04-13 \
  --base-type hire \
  --attendance-rate 1.0
```

### 2. 미사용 연차수당 (`unused-pay`)

통상임금 일액 + 미사용일수 입력 → 수당 산정. 사용촉진 이행 여부 반영.

```bash
python3 skills/annual-leave/references/calculator.py unused-pay \
  --daily-ordinary-wage 100000 \
  --unused-days 5 \
  --promotion-completed false
```

## 입력 파싱 가이드

사용자가 정보를 일부만 줘도 우선 `entitlement` 모드로 추정 후, 수당 계산 필요 시 추가 정보 요청.

| 사용자 입력                    | 모드        | 누락 시 기본값                      |
| ------------------------------ | ----------- | ----------------------------------- |
| "입사일 2022-01-01, 오늘 기준" | entitlement | base-type=hire, attendance-rate=1.0 |
| "연차 5일 남았어, 일급 10만"   | unused-pay  | promotion-completed=false           |
| "연차 얼마나 남아?"            | 되묻기 1번  | 입사일 + 기준일(또는 근속년수)      |

## 되묻기 규칙 (정보 부족 시)

1. **입사일 부족**: "입사일(YYYY-MM-DD)을 알려주세요."
2. **기준일 부족**: "기준 날짜(오늘 또는 특정 날짜)를 알려주세요. 기본값은 오늘로 계산합니다."
3. **출근율 불명확**: "출근율이 80% 미만인 경우가 있으신가요? 없으면 기본값(100%)으로 계산합니다."
4. **회계연도 vs 입사일 기준**: "사업장이 회계연도(1/1 또는 4/1 등) 기준으로 일괄 관리하나요, 개인 입사일 기준인가요?"
5. **수당 계산**: "통상임금 1일분(원)과 미사용 연차일수를 알려주세요."

## 응답 포맷

계산 결과 + 다음 항목 안내:

- 근속연수 / 기준유형
- 연차 발생일수 (가산내역 포함)
- 미사용 연차수당 (해당 시)
- 사용촉진 이행 여부 및 면제 사유
- 산식 (검증용)
- 면책: "단체협약·취업규칙에 따라 실제 일수가 달라질 수 있습니다."

## 알려진 한계

1. **출근율 80% 미만 세부 계산 미지원** — 출근율 미달 시 발생일수 0일 처리. 결근 사유별 산입 여부(산재·출산휴가 등)는 별도 검토 필요.
2. **회계연도 기준 중간 연도 비례 미지원** — 첫해 비례 계산만 지원. 중도 퇴사 시 잔여 일수 비례정산은 별도 산정.
3. **연차 소멸 시효** — 1년간 미사용 연차는 소멸 원칙 (§60⑦). 본 계산기는 소멸 이력 미추적.
4. **시간 단위 연차** — §60⑧ 시간 단위 연차 사용 미지원.
5. **대체휴가·보상휴가** — §57 보상휴가제 적용 사업장은 별도 확인 필요.

## 관련 스킬

- `severance-pay` — 퇴직금 산정 (미사용 연차수당을 평균임금에 포함 여부 검토)
- `wage-base` — 통상임금 산정 베이스 (연차수당 1일분 계산 기초)
- `working-hours` — 연차 사용 시 소정근로일수·출근율 산정
