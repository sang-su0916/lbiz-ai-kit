# 주휴수당 계산기 (weekly-holiday-pay)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **근로기준법 §55①** — 사용자는 근로자에게 1주 평균 1회 이상의 유급휴일(주휴일)을 부여해야 한다
- **근로기준법 시행령 §30** — 1주 소정근로시간 15시간 미만인 단시간 근로자는 주휴수당 적용 제외
- **5인 미만 사업장도 적용** — §55 주휴일은 상시 근로자 수에 관계없이 적용 (§56 연장·야간·휴일 가산수당과 다름)

## 발생 요건 (2가지 모두 충족 필요)

| 요건             | 기준            | 비고                       |
| ---------------- | --------------- | -------------------------- |
| 1주 소정근로시간 | **15시간 이상** | 시행령 §30                 |
| 소정근로일 개근  | **결근 없음**   | 조퇴·지각은 개근 방해 아님 |

## 핵심 공식

```
[주 40시간 이상]
주휴수당 = 통상시급 × 8시간

[주 15~39시간 (단시간 근로자 비례)]
주휴수당 = 통상시급 × (주 소정근로시간 ÷ 40) × 8시간

[주 15시간 미만]
주휴수당 미발생

[월 환산]
월 주휴수당 = 주휴수당 × 4.345주
```

## 두 가지 모드

### 1. 발생 여부 판정 (`eligibility`)

소정근로시간 + 개근 여부만으로 발생 여부 확인.

```bash
python3 skills/weekly-holiday-pay/references/calculator.py eligibility \
  --weekly-hours 40 \
  --worked-all-days
```

### 2. 금액 산정 (`calculate`)

발생 여부 판정 + 주휴수당 금액 + 월 환산까지 한 번에.

```bash
python3 skills/weekly-holiday-pay/references/calculator.py calculate \
  --weekly-hours 40 \
  --hourly-wage 10320 \
  --worked-all-days
```

## 입력 파싱 가이드

| 사용자 입력                           | 모드        | 누락 시 기본값                |
| ------------------------------------- | ----------- | ----------------------------- |
| "주 40시간, 시급 10,320원, 개근"      | calculate   | —                             |
| "주 20시간, 시급 12,000원, 결근 없음" | calculate   | —                             |
| "주휴수당 받을 수 있나요? 주 15시간"  | eligibility | worked-all-days=false         |
| "주휴수당 얼마?"                      | 되묻기 1번  | 소정근로시간, 시급, 개근 여부 |

## 되묻기 규칙 (정보 부족 시)

1. **소정근로시간 부족**: "1주 소정근로시간이 몇 시간인지 알려주세요."
2. **개근 여부 불명확**: "해당 주에 결근 없이 모두 출근하셨나요? (조퇴·지각은 개근으로 인정됩니다.)"
3. **시급 부족** (금액 산정 시): "통상시급이 얼마인지 알려주세요. 월급제라면 월급 ÷ 209시간으로 추산할 수 있습니다."

## 응답 포맷

계산 결과 + 다음 항목 안내:

- 발생 여부 (eligible: true/false)
- 비례 지급 시간수 (paid_hours)
- 주휴수당 금액 (원)
- 월 환산 금액 (4.345주 기준)
- 산식 (검증용)
- 미발생 시 사유 명시
- 체불 안내: "주휴수당 미지급은 임금체불로 고용노동부 진정 대상입니다."

## 알려진 한계

1. **결근 1회만 있어도 그 주 주휴수당 미발생** — 조퇴·지각은 개근 인정이나 결근은 1회도 허용 안 됨. 주 단위로 판정.
2. **소정근로일 기준** — 계약상 소정근로일 개근 여부가 기준. 추가 연장근무 일수는 무관.
3. **통상임금 범위 미반영** — 식대·교통비 등 통상임금 포함 여부 분쟁 시 `wage-base` 스킬 참조.
4. **월급제 시급 환산** — 월급 ÷ 209시간(주 40시간 기준)은 추산값. 실제 소정근로시간에 따라 달라질 수 있음.

## 관련 스킬

- `minimum-wage` — 최저임금 위반 여부 (시급 기준 확인)
- `overtime-pay` — 연장·야간·휴일 가산수당 산정
