# 통상임금·평균임금 계산기 (wage-base)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

- **근로기준법 §2①6** — 통상임금: 소정근로 또는 총근로에 대하여 정기적·일률적·고정적으로 지급하기로 정한 시간급·일급·주급·월급 금액 또는 도급 금액
- **근로기준법 §2①6** — 평균임금: 산정사유 발생일 직전 3개월간 근로자에게 지급된 임금 총액 ÷ 그 기간의 총 일수
- **근로기준법 §2②** — 산출된 평균임금이 통상임금보다 적은 경우 통상임금을 평균임금으로 봄
- **대법원 2013다89399(전합)** — 통상임금의 정기성·일률성·고정성 기준 재정립

## 핵심 공식

### 통상임금 환산

```
209시간 근거: (40시간 + 유급주휴 8시간) × 365일/12개월/7일 = 209.143... → 209시간 적용
  = 주 40시간 × 4.345주/월 + 주휴 8시간 × 4.345주/월 = 173.8 + 34.7 ≈ 208.5 → 209시간

시급 = 월급 ÷ 209
일급 = 시급 × 8  (1일 소정근로시간 8시간)
월급 = 시급 × 209
```

### 평균임금 산정

```
1일 평균임금 = (직전 3개월 임금총액 + 연간상여금×3/12 + 연차수당×3/12) ÷ 3개월 일수
```

### 통상 vs 평균 비교 (근로기준법 §2②)

```
적용 일급 = max(통상임금 일급, 평균임금 일급)
```

## 서브커맨드 3개

### 1. `ordinary` — 통상임금 환산

월급·시급·일급 중 하나를 입력하면 나머지 단위로 모두 환산.

```bash
python3 skills/wage-base/references/calculator.py ordinary \
  --base-wage 3000000 \
  --base-type monthly \
  --weekly-hours 40
```

```bash
python3 skills/wage-base/references/calculator.py ordinary \
  --base-wage 12000 \
  --base-type hourly
```

### 2. `average` — 평균임금 산정

직전 3개월 임금 총액과 일수를 입력하면 1일 평균임금 산출.

```bash
python3 skills/wage-base/references/calculator.py average \
  --three-month-wage-total 10500000 \
  --days-in-3month 92 \
  --annual-bonus 0 \
  --annual-leave-pay 0
```

### 3. `compare` — 통상 vs 평균 비교

퇴직금·휴업수당 산정 시 두 일급 중 큰 쪽 판정.

```bash
python3 skills/wage-base/references/calculator.py compare \
  --ordinary-daily 114832 \
  --average-daily 100000
```

## 입력 파싱 가이드

| 사용자 입력                | 서브커맨드                   | 누락 시 기본값                          |
| -------------------------- | ---------------------------- | --------------------------------------- |
| "월급 300만, 시급 환산"    | ordinary --base-type monthly | weekly-hours=40                         |
| "시급 12,000원, 월급 환산" | ordinary --base-type hourly  | weekly-hours=40                         |
| "직전 3개월 1050만, 92일"  | average                      | bonus=0, leave_pay=0                    |
| "통상 vs 평균 어느 쪽?"    | compare                      | ordinary-daily, average-daily 모두 필요 |

## 되묻기 규칙

1. **임금 금액 부족**: "월급, 시급, 일급 중 어느 기준으로 알고 계신가요?"
2. **base-type 불명확**: "금액이 월급 기준인가요, 시급 기준인가요?"
3. **평균임금 일수 부족**: "직전 3개월 일수를 알려주세요 (보통 89~92일)."
4. **compare 입력 부족**: "통상임금 일급과 평균임금 일급을 모두 알려주세요."

## 응답 포맷

계산 결과 + 다음 항목 안내:

- 입력 기준 (monthly/hourly/daily) 및 금액
- 209시간 환산 근거 명시
- hourly / daily / monthly 환산 결과
- 평균임금 산정 시: 1일 평균임금 + 산식
- 비교 시: 적용 기준 (통상 or 평균) + 이유
- 면책: "단체협약·취업규칙·소정근로시간에 따라 실제 금액이 달라질 수 있습니다."

## 검증 예시

| 입력                                 | 결과                       |
| ------------------------------------ | -------------------------- |
| 월급 3,000,000 → 시급                | 3,000,000 ÷ 209 = 14,354원 |
| 월급 3,000,000 → 일급                | 14,354 × 8 = 114,832원     |
| 시급 12,000 → 월급                   | 12,000 × 209 = 2,508,000원 |
| 3개월 10,500,000 / 92일              | 일평균 114,130원           |
| 통상일급 114,832 vs 평균일급 100,000 | 통상임금 적용              |

## 알려진 한계

1. **주휴수당 포함 여부** — 본 계산은 주 40시간 기준 209시간 적용. 단시간근로자(주 15시간 미만)는 주휴 미발생, 별도 산식 필요.
2. **통상임금 범위 판단 불가** — 어떤 수당이 통상임금에 해당하는지(정기성·일률성·고정성)는 법적 판단 영역. 본 계산기는 사용자가 제공한 금액 기준으로만 환산.
3. **연장·야간·휴일 가산율 미포함** — 수당 금액 산정(통상임금 × 가산율)은 별도 계산 필요.
4. **임금 구성항목 분리 미지원** — 기본급, 식비, 교통비 등 항목별 통상임금 해당 여부 판단 불포함.
