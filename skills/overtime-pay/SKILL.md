# 연장·야간·휴일 가산수당 계산기 (overtime-pay)

⚠️ **반드시 계산기 CLI를 실행하세요. 직접 계산 금지** — hallucination 방지.

## 법령 근거

| 조문                                  | 내용                               | 가산율                   |
| ------------------------------------- | ---------------------------------- | ------------------------ |
| **근기법 §56①**                       | 연장근로 (1일 8H / 1주 40H 초과)   | 통상임금의 **50% 이상**  |
| **근기법 §56② 전단** (2018.3.20 개정) | 휴일근로 8시간 이내                | 통상임금의 **50% 이상**  |
| **근기법 §56② 후단** (2018.3.20 개정) | 휴일근로 8시간 초과분              | 통상임금의 **100% 이상** |
| **근기법 §56③**                       | 야간근로 (오후 10시~익일 오전 6시) | 통상임금의 **50% 이상**  |

> **2018.3.20 개정 이력**: 개정 전 구법에서는 휴일근로에 대해 단순 50% 가산만 명시했으나,  
> 개정으로 8시간 초과분에 100% 가산이 명문화되었습니다.

### 대법원 판례 근거 (중복 가산)

- **대법 2011다112247** (2014.6.26 전합): 휴일근로와 야간근로가 겹치는 구간은 **휴일가산 + 야간가산을 각각 독립 합산** (50%+50%=100% 추가).
- **대법 2013다109527** (2018.1.25 전합): 휴일근로 중 8시간 초과분은 연장근로 가산을 별도 중복 적용하지 않는다는 취지로 정리(구법 기간). 현행 §56② 후단(2018.3.20 개정)으로 **100% 가산을 명문화**해 같은 결론을 입법 확정.
- 실무 요지: 휴일 8H 초과분은 **휴일(50%)+연장(50%)=100% 가산**이고, 여기에 야간이 겹치면 **+50%** 더 붙어 총 150% 가산(=2.5배 배율).

## 가산율 상수

```python
OVERTIME_PREMIUM      = 0.50   # 연장근로 50%
NIGHT_PREMIUM         = 0.50   # 야간근로 50%
HOLIDAY_PREMIUM       = 0.50   # 휴일근로 8시간 이내 50%
HOLIDAY_OVER_8H_PREMIUM = 1.00 # 휴일근로 8시간 초과분 100%
NIGHT_START_HOUR      = 22     # 오후 10시
NIGHT_END_HOUR        = 6      # 익일 오전 6시
LEGAL_DAILY_HOURS     = 8      # 법정 1일 근로시간
LEGAL_WEEKLY_HOURS    = 40     # 법정 1주 근로시간
```

## 최종 지급액 산식

```
최종 지급액 = 통상임금(원금) + 가산액
원금 = 통상시급 × 근로시간
가산액 = 통상시급 × 각 유형별 시간 × 가산율
```

## 중복 가산 원칙

야간근로는 연장·휴일 여부와 무관하게 **독립적으로 추가 가산**합니다.

| 근로 유형                      | 원금 배율 | 가산     | 총 배율   |
| ------------------------------ | --------- | -------- | --------- |
| 연장근로만                     | 1.0       | +0.5     | **1.5배** |
| 야간근로만                     | 1.0       | +0.5     | **1.5배** |
| 휴일근로 (8H 이내)만           | 1.0       | +0.5     | **1.5배** |
| 휴일근로 (8H 초과분)만         | 1.0       | +1.0     | **2.0배** |
| 연장 + 야간 (겹침)             | 1.0       | +0.5+0.5 | **2.0배** |
| 휴일 (8H 이내) + 야간 (겹침)   | 1.0       | +0.5+0.5 | **2.0배** |
| 휴일 (8H 초과분) + 야간 (겹침) | 1.0       | +1.0+0.5 | **2.5배** |

### 계산 예시 (시급 10,000원 기준 — 암산 검증용)

1. **평일 연장 2H**: 10,000×2×1.5 = **30,000원**
2. **평일 연장 2H + 야간 2H 겹침**: 10,000×2×(1.5+0.5) = **40,000원** _(시급 15,000원이면 60,000원)_
3. **휴일 6H**: 10,000×6×1.5 = **90,000원**
4. **휴일 10H (8H 이내 + 초과 2H)**: 10,000×8×1.5 + 10,000×2×2.0 = 120,000 + 40,000 = **160,000원**
5. **휴일 8H 중 야간 2H**: 10,000×8×1.5 + 10,000×2×0.5 = 120,000 + 10,000 = **130,000원**
6. **휴일 10H + 야간 2H (가장 복잡, 휴일+연장+야간 중복)**:  
   휴일 8H 이내 120,000 + 휴일초과 2H 40,000 + 야간 2H 10,000 = **170,000원**

> 위 6개 시나리오는 `calculator.py calculate` 모드에서 파라미터만 조정해 그대로 재현됩니다.

## 5인 미만 사업장 적용 제외

**근로기준법 시행령 §7 [별표]** — 상시 5인 미만 사업장은 §56 가산수당 규정 **적용 제외**.  
이 경우 통상임금 원금만 지급하며, 별도 안내 메시지를 출력합니다.

> ⚠️ 사업장 규모를 반드시 확인하세요. 5인 미만 사업장 근로자는 법정 가산수당 청구권이 없습니다.

## 적용 제외 대상 (알려진 한계)

- **§63 관리·감시·단속적 근로자**: 농림·수산업 종사자, 감시·단속적 근로에 종사하는 자로서 고용노동부장관 승인을 받은 경우 §56 적용 제외.
- **포괄임금제**: 가산수당을 포함한 포괄임금 약정이 있는 경우, 실제 초과근로 대비 포괄임금이 적으면 그 차액 청구 가능 (대법원 판례). 본 계산기는 포괄임금 유효성 검토 미지원.
- **선택적 근로시간제·탄력적 근로시간제**: 정산기간 내 평균으로 연장근로 여부를 판단하므로 본 계산기와 산정 방식 상이. 별도 계산 필요.
- **유연근로시간제 특례업종**: 특례업종(§59) 근로자는 연장근로 한도 특례 적용. 본 계산기 미지원.

## CLI 사용법

### `calculate` — 복합 가산수당 계산

```bash
python3 skills/overtime-pay/references/calculator.py calculate \
  --hourly-wage 10320 \
  --overtime-hours 2 \
  --night-hours 0 \
  --holiday-hours 0 \
  --holiday-over-8-hours 0 \
  --company-size-ge-5
```

5인 미만 사업장:

```bash
python3 skills/overtime-pay/references/calculator.py calculate \
  --hourly-wage 10320 \
  --overtime-hours 2 \
  --no-company-size-ge-5
```

### `breakdown` — 출퇴근 시각 기반 시간 분해

```bash
python3 skills/overtime-pay/references/calculator.py breakdown \
  --clock-in "09:00" \
  --clock-out "23:30" \
  --break-minutes 60
```

휴일 근무:

```bash
python3 skills/overtime-pay/references/calculator.py breakdown \
  --clock-in "09:00" \
  --clock-out "23:30" \
  --is-holiday \
  --break-minutes 60
```

## 입력 파싱 가이드

| 사용자 입력               | 액션                                          | 누락 시 기본값     |
| ------------------------- | --------------------------------------------- | ------------------ |
| "시급 1만원, 연장 2시간"  | `calculate` 즉시 실행                         | night=0, holiday=0 |
| "오전 9시~오후 11시 30분" | `breakdown` → `calculate`                     | break=60분, 평일   |
| "휴일에 10시간 일했어요"  | `calculate` holiday-hours=10 holiday-over-8=2 | 사업장 규모 되묻기 |
| 시급 정보 없음            | 되묻기                                        | —                  |

## 되묻기 규칙 (정보 부족 시)

1. **시급 정보 없음**: "통상시급(원)을 알려주세요. 월급을 월 근로시간으로 나눈 값입니다."
2. **근무 시간대 불명확**: "연장·야간·휴일 각각 몇 시간인지, 또는 출퇴근 시각을 알려주세요."
3. **휴일 여부 불명확**: "주휴일·법정공휴일·약정휴일 등 휴일 근무인가요?"
4. **야간 시간대 확인**: "근무 시간에 밤 10시~다음날 오전 6시 구간이 포함되나요? 몇 시간인가요?"
5. **사업장 규모 불명확**: "상시 근로자 수가 5인 이상인가요? (5인 미만이면 가산수당 적용 제외)"

## 응답 포맷

계산 결과에 다음 항목 포함:

- 항목별 {원금, 가산액, 소계}
- 총 지급액 (grand_total)
- 적용 가산율 및 중복 가산 여부
- 5인 미만 시 안내 메시지
- 면책: "통상임금 범위·포괄임금 약정·단체협약에 따라 실제 금액이 달라질 수 있습니다."

## 관련 스킬

- `wage-base` — 통상임금 구성 항목 산정 (기본급, 고정수당 포함 여부)
- `weekly-holiday-pay` — 주휴수당 계산
- `severance-pay` — 퇴직금 산정 (평균임금 기반)
